The Vulnerability Database 

SCAP - Security Content Automation Protocol - method for standardizing and automating the vulnerability management and policy compliance in your organization  

- Protocol that can be used by a vuln scanner to automatically determine if a host meets some config baseline 
    - Does the host pass the necc test? 
    - Is that host secure enough for us? 
        
- CVE - common vuln and exposures (MITRE) 
- CPE - common platform enumeration - describes os, app, hardware devices etc.
